# 🎬 Netflix Movie Analysis

## 📌 Overview

This project is based on analyzing Netflix movie data to understand trends in genres, ratings, and content over time.
The goal is to explore the dataset and extract useful insights using Python libraries.

## 📊 Dataset Information

* The dataset contains information about Netflix movies.
* Main features include:

  * Title
  * Genre
  * Rating
  * Release Year
  * Duration

## 🛠️ Tools & Technologies

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn

## 🔍 Analysis Performed

* Checked data quality (missing values, duplicates)
* Explored distribution of movie genres
* Analyzed ratings across different genres
* Studied how movie releases have changed over the years

## 📈 Key Insights

* Some genres like Drama and Comedy appear more frequently
* Ratings are not evenly distributed across all genres
* There is a noticeable increase in the number of movies released in recent years
